**Why:** Usuário solicitou redesign profissional nível enterprise com paleta elegante, design limpo e flows operacionais para o Brasil.

**How to apply:** Sempre usar as classes utilitárias do design system (page-header, page-title, page-subtitle, kpi-card, badge-*, nav-section-label, card-interactive) ao criar novas páginas. Manter padrão navy sidebar + white content + blue primary.
